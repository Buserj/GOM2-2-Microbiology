GitHub for useful scripts in analyzing the UT GOM2-2 2023 Expedition Microbiological Data
Buser-Young et al., In Prep. JGR Biogeosciences

Raw sequencing data can be found at NCBI Repository Number XXX and were analyzed using QIIME2 v.2024.5.


**Phylogenetic_Analysis.R**
Which calculates alpha (net relatedness index, faith's diversity, and nearest taxon index) and 
beta (beta nearest taxon index and raup-crick bray) phylogenetic diversity.

**Contamination_Analysis.R**
Demonstrates stringent negative control filtering and contaminant taxa assessment.
